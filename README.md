#### 拉取代码方法

2. 然后选择点击git bash here打开命令行窗口

3. 复制以下命令

4. 注意不要使用Ctrl+V粘贴，否则会出现错误
   正确粘贴快捷键是Shift+Ins,或者右键点击再选Paste

5. 按Enter执行即可拉取代码
